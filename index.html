Q1. give output.

Ans:-

        "let count = 0;
        (function () {
          if (count === 0) {
            let count = 1;
            console.log(count); // What is logged?
          }
          console.log(count); // What is logged?
        })();"

Output:-1,0

explanation:-

The code initializes a variable "count" as 0. Within a function, if "count" equals 0, a new local variable "count" is created with a value of 1, and it is printed. Later, the original "count" variable is printed. The purpose is to demonstrate variable scoping in JavaScript. The output will be 1 followed by 0, illustrating that the local variable inside the function does not affect the outer variable.

Q3.Write a code to calculate area of a rectangle using inner function. In this case outer function should accept parameter length and inner function should accept parameter breadth.

Ans:-

        function calculateRectangleArea(length) {
            function calculateArea(breadth) {
              return length * breadth;
            }
            
            return calculateArea;
          }
          
          const rectangleArea = calculateRectangleArea(5);
          console.log(rectangleArea(8)); // Output: 40

Output:-3 3 3

explanation:-

In this code, the outer function calculateRectangleArea accepts the parameter length and defines an inner function calculateArea that accepts the parameter breadth. The inner function multiplies the length and breadth to calculate the area. The outer function returns the inner function calculateArea itself. By invoking calculateRectangleArea with a specific length value, we get the calculateArea function as the result. We can then call calculateArea with the desired breadth value to calculate the area of the rectangle

Q5.Take a variable in outer function and create an inner function to increase the counter every time it is called

Ans:-

        function counter() {
            let count = 0;
          
            function increaseCounter() {
              count++;
              console.log(count);
            }
          
            return increaseCounter;
          }
          
          const increase = counter();
          increase(); // Output: 1
          increase(); // Output: 2
          increase(); // Output: 3

Output:-1 2 3

explanation:-

In this code, the outer function counter initializes a variable count to 0. It defines an inner function increaseCounter that increments the count variable by 1 and logs its value to the console. The outer function then returns the inner function increaseCounter. By invoking the outer function and assigning it to the increase variable, we obtain the increaseCounter function. Each time we call increase(), it increases the counter and logs the updated value.
